Demonstrates 50 different graphics animations in DOS.

It was developed by combining various mathematical formulae is various (random)
ways. Observing that interesting animations can be produced in such ways, this
application was developed. This is a demo and can be used as an insipartion as
to what kinds of art can be performed with Math.
